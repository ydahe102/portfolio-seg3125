# Yasmine Bachir - Design Portfolio

This is my personal design portfolio created for SEG3125 - Analysis and Design of User Interfaces at the University of Ottawa. It showcases who I am, how I work, and four upcoming case studies.

🔗 **Live site:** [https://ydahe102.netlify.app](https://ydahe102.netlify.app)

---

## About This Project

This portfolio is my first time coding a design from scratch. I have experience designing SharePoint pages, but this project marks my transition into building interfaces with code using HTML, CSS, JavaScript, and React.

The site includes:
- An **about section** with my photo and background
- A **"How I Work"** section explaining my experience and workflow
- **Four project cards** linking to hold pages for future case studies
- A **contact section** with email, GitHub, and LinkedIn links

---

## Technologies Used

| Technology | Purpose |
|------------|---------|
| React | Component-based UI framework |
| React Router DOM | Navigation between pages |
| Bootstrap 5 | Grid system, navbar, cards, and responsive utilities |
| Bootstrap Icons | Iconography across the site |
| CSS3 | Custom styling and theme overrides |

## Color Palette

The portfolio uses a warm dusty pink theme that matches my personal style and photo:

| Color | Hex | Usage |
|-------|-----|-------|
| Dusty Pink | `#D4A5A5` | Buttons, links, icons, name highlight |
| Warm Taupe | `#5C4B3A` | Body text |
| Soft Taupe | `#8E7C6A` | Secondary text |
| Warm Cream | `#FFF9F5` | Page background |
| Soft Blush | `#F8EDED` | Icon backgrounds |
| Soft Peach-Beige | `#FDF5F0` | Section backgrounds |
